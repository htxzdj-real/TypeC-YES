# Mini-USB 转接板

## 测试中-- 外观预计还行，牢固程度 ≈ 原本 Mini-USB

拆掉原本的插座，使用一个底部是镜像 Mini-USB 焊盘的转接板在原位置进行转换


注意：本板不可以逆转换，有短路&反接风险
